# Plataforma Online: IFRS

## Curso: Administrador de Banco de Dados

Este curso abrangente oferece uma introdução aos fundamentos essenciais dos Bancos de Dados e aborda tópicos importantes para o sucesso na área. Durante o curso, você aprenderá os conceitos fundamentais de Bancos de Dados, incluindo estruturas de dados, modelagem relacional, normalização e indexação. Além disso, você receberá treinamento prático em SQL, a linguagem de consulta padrão para interagir com Bancos de Dados. Através de exercícios práticos, você ganhará experiência em manipulação de dados, criação de tabelas, consultas complexas e muito mais. Também abordaremos especificamente a administração de Bancos de Dados PostgreSQL e MySQL, onde você aprenderá a configurar e otimizar o desempenho desses sistemas, além de gerenciar segurança, backup e recuperação de dados.
